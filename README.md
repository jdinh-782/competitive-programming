# Competitive Programming
A repository to hold practice and test questions for competitive programming.


## Overview
This project is the main source hub for storing practice and test questions that I use to prepare myself for competitive programming and leetcode-style problems. I typically use Python3 in my solutions, but there may be other languages that I might use depending on the question.


## Setup and Installation
Clone the repository into an empty directory. Then, checkout to the main branch and pull the latest changes. <br>
Run the following commands in a terminal:
```bash
git clone https://github.com/jdinh-782/competitive-programming.git
git checkout main
git pull
```


## Build, Test, and Run
Depending on your programming language configuration, you should be able to run the individual files (problems) on your local device.
